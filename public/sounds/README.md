# Alert Sound Files

This directory contains audio files for different fraud alert levels:

## Sound Files:
- `low-risk.mp3` - Gentle notification sound for low-risk alerts
- `medium-risk.mp3` - Attention-grabbing sound for medium-risk alerts  
- `high-risk.mp3` - Urgent warning sound for high-risk alerts
- `critical-risk.mp3` - Alarm-like sound for critical threats
- `success.mp3` - Positive confirmation sound
- `error.mp3` - Error notification sound

## Audio Specifications:
- Format: MP3
- Quality: 128kbps
- Duration: 1-3 seconds each
- Volume: Normalized for consistent playback

## Usage:
These sounds are played by the AlertPopup component based on the detected threat level.
